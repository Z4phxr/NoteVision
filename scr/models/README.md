# Models

This package contains domain models for page metadata and image slices.

## Models
- `PageRecord`: metadata for one page, plus staff slices.
- `StaffSlice`: one 5-line staff slice from a page.
- `ChunkSlice`: vertical chunk from a staff slice.
- `PageDataset`: container for page records.
- `DatasetManager`: helper for attaching slices to a dataset.
